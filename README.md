# Elite Group Next.js Site

## Run locally

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Production build

```bash
npm run build
npm run start
```

## Notes

- Replace placeholder contact details in `app/layout.js` and `app/contact/page.js`
- Replace placeholder copy in `components/data.js`
- Replace gradient blocks with real images if you want a more finished look
